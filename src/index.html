Как работает мой SDR?

Работает на одноплатном комптютере
В качестве сервера используется одноплатный компьютер orangePi PC 2. Ресурсы позволяют использовать до 3х человер одновременно.

Работа только на одном участке

Используется только один свисток SDR. Из - за этого можно слушать только один диапазон. Если переключить на другой, остальные слушатели переходят на него автоматически.

Aналоговые и цифровые виды радиосвязи
Можно слушать цифру FT4, FT8, DRM, APRSpacket, FST4... Аналоговая связь FM, WFM, AM, LSB, USB, CW...

Использую антенны КВ и УКВ
Антенны на УКВ 2м-0.7м J-антенна. На КВ длинный луч

pskreporter.info

OpenWebRX используется автоматические отправки связей на сервер. На текущий момент работает только pskreporter.info с протоколом связи FT8. Отправка происходит как только компьютер дешифрует связь
Как работает спот
1. перейти на нужный диапазон.
2. выбрать в меню DIG FT8.
3. Найти в окне свой позывной.
4. Перейти на сайт pskreporter.info.
5. Найти мой позывной EW8MKU
